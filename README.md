# Our Family Trip to Italy · September 2027

A single-page travel guide for our family trip to Florence & Rome (Sept 1–12, 2027).

Built as one self-contained `index.html` file: day-by-day itinerary, an interactive
map of every restaurant/museum/winery, budget breakdown, a tickable pre-trip
checklist and packing list, what-to-wear notes, Italian phrases with spoken
pronunciation, and direct booking links.

## View it

Open `index.html` in any browser, or visit the live site (GitHub Pages).

## Offline map

`italy-map.kml` can be imported into [Google My Maps](https://mymaps.google.com)
(Create a new map → Import) to use all the pins in the Google Maps app with
offline navigation.
